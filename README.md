## üì¶ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aniruddha-10/DevPortfolio.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run locally:**
   ```bash
   npm start
   ```
   The site will start at [http://localhost:3000](http://localhost:3000) (or another port).
